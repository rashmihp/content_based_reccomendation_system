 <p>This book is an introduction to programming in <a rel="nofollow" class="external text" href="http://www.oracle.com/us/technologies/java/index.htm">Oracle’s Java™ programming language</a>, a widely used programming language and software platform. This book serves as a comprehensive guide, complete with a <a href="/wiki/Java_Programming/Tutorials" title="Java Programming/Tutorials">series of tutorials</a> to help users better understand the many ways one can program in Java. In its entirety, this book is meant to be both an introductory guide and a useful reference on Java and related technologies. As is the nature of this book, the content within the book is continuously being updated and revised. With every chapter within this book, the complexity of the context increases, building up on lessons learnt in the previous chapters. Beginners should therefore start from the beginning and proceed in a sequence throughout the material of the book. </p> 
 <p>If you are not familiar with object-oriented programming, you should first read the book <a href="/wiki/Object_Oriented_Programming" title="Object Oriented Programming">Object Oriented Programming</a>. Other special purpose books on Java, such as <a href="/wiki/Java_EE_Programming" title="Java EE Programming">Java Enterprise Edition programming</a>, <a href="/wiki/Java_Persistence" title="Java Persistence">Java Persistence</a>, or <a href="/wiki/Java_Swings" title="Java Swings">Java Swing programming</a>, with more details, would complement this book nicely. </p>
